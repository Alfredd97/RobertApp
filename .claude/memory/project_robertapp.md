Full-stack artist portfolio site for "Robert Avery" built with Next.js 14 App Router, Tailwind CSS, TypeScript, and Anthropic SDK streaming chat.

**Why:** User requested a complete artist landing page with AI customer service widget.

**How to apply:** When working in this repo, refer to CLAUDE.md for commands, design tokens, and customization points. The chat widget streams from `/api/chat` using `claude-sonnet-4-6` with prompt caching on the system message.
